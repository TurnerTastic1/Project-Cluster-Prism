## API/Server Usage

Clone [Project-Cluster-Prism](https://github.com/TurnerTastic1/Project-Cluster-Prism)
#### To start:
npm run clean
npm run build
npm run start

#### To enter dev mode:
npm run clean
npm run build
npm run dev
